# PhantomAPI 架构设计

## 整体架构

```
┌─────────────────────────────────────────────────────────────────────┐
│                         控制端 (PC/手机)                             │
│                     HTTP Client (curl/Python/JS)                    │
└─────────────────────────────────┬───────────────────────────────────┘
                                  │ HTTP :9999
                                  ▼
┌─────────────────────────────────────────────────────────────────────┐
│                        PhantomAPI 应用                               │
├─────────────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                    HttpServerEngine                          │   │
│  │                    (NanoHTTPD :9999)                         │   │
│  │  ┌──────────┬──────────┬──────────┬──────────┬──────────┐  │   │
│  │  │/api/sys  │/api/ui   │/api/web  │/api/cdp  │/api/net  │  │   │
│  │  └──────────┴──────────┴──────────┴──────────┴──────────┘  │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                  │                                   │
│  ┌───────────────────────────────┴───────────────────────────────┐ │
│  │                    PhantomAccessibilityService                 │ │
│  │  - UI 树遍历 (AccessibilityNodeInfo)                           │ │
│  │  - 坐标点击/滑动                                               │ │
│  │  - 节点操作 (click/scroll/setText)                            │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────┐
│                        LSPosed Hook 层                               │
├─────────────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                      WebViewHook                              │   │
│  │  - Hook WebViewClient.onPageFinished → 注入 JS               │   │

│  │  - 写入 DOM 到 /data/local/tmp/phantom/dom.json              │   │
│  └─────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────┘
                                  │
                                  ▼
┌─────────────────────────────────────────────────────────────────────┐
│                        目标应用进程                                  │
│  ┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐    │
│  │   WebView 页面   │  │   Chrome 页面   │  │   原生 UI 页面   │    │

│  └─────────────────┘  └─────────────────┘  └─────────────────┘    │
└─────────────────────────────────────────────────────────────────────┘
```

## WebView DOM 提取流程

```
┌──────────────────────────────────────────────────────────────────┐
│                      WebView 页面加载完成                          │
└─────────────────────────────┬────────────────────────────────────┘
                              │
                              ▼
┌──────────────────────────────────────────────────────────────────┐
│  LSPosed Hook: WebViewClient.onPageFinished()                    │
│  - 检查 URL 是否为 http/https                                    │
│  - 防重复注入 (通过 WebView.setTag)                              │
└─────────────────────────────┬────────────────────────────────────┘
                              │
                              ▼
┌──────────────────────────────────────────────────────────────────┐
│  注入 JS 脚本 (evaluateJavascript)                               │
│  - 遍历 DOM: document.querySelectorAll('body *')                 │
│  - 提取文本、坐标、尺寸、标签                                     │
│  - 通过 prompt('phantom://dom', JSON.stringify(data)) 发送       │
└─────────────────────────────┬────────────────────────────────────┘
                              │
                              ▼
┌──────────────────────────────────────────────────────────────────┐
│  LSPosed Hook: WebChromeClient.onJsPrompt()                      │
│  - 拦截 message === 'phantom://dom'                              │
│  - 从 args[3] 获取 defaultValue (JSON 数据)                      │
│  - 返回 true 阻止弹窗                                            │
└─────────────────────────────┬────────────────────────────────────┘
                              │
                              ▼
┌──────────────────────────────────────────────────────────────────┐
│  写入文件 (异步线程)                                              │
│  - 路径: /data/local/tmp/phantom/dom.json                        │
│  - 权限: 777 (所有进程可读写)                                    │
└─────────────────────────────┬────────────────────────────────────┘
                              │
                              ▼
┌──────────────────────────────────────────────────────────────────┐
│  API: GET /api/web/dom                                           │
│  - 检查文件新鲜度 (< 30秒)                                       │
│  - 读取文件内容                                                   │
│  - 删除文件 (防止脏数据)                                         │
│  - 返回 JSON 数组                                                │
└──────────────────────────────────────────────────────────────────┘
```

## 文件 IPC 机制

```
┌─────────────────────────────────────────────────────────────────┐
│                    /data/local/tmp/phantom/                     │
├─────────────────────────────────────────────────────────────────┤
│  dom.json          - WebView DOM 数据 (LSPosed 模块写入)        │
│                                                                 │
│  权限: drwxrwxrwx (777) - 所有进程可读写                        │
│                                                                 │
│  生命周期:                                                       │
│  1. WebView 页面加载完成 → JS 提取 DOM                          │
│  2. LSPosed Hook 写入 dom.json                                  │
│  3. API 读取后删除 (用完即焚)                                    │
│  4. 下次页面加载时重新生成                                       │
└─────────────────────────────────────────────────────────────────┘
```

## API 设计原则

1. **RESTful 风格**
   - GET 用于查询操作
   - POST 用于修改操作
   - JSON 响应格式

2. **错误处理**
   - 成功: `{"success": true, ...}`
   - 失败: `{"error": "error_code", "hint": "提示信息"}`

3. **新鲜度控制**
   - DOM 数据 30 秒过期
   - 防止读取脏数据

4. **节流保护**
   - 高频操作 (tap/swipe) 100ms 节流
   - 防止误操作

## 安全考虑

1. **局域网限制**
   - 仅监听局域网 IP
   - 不暴露到公网

2. **权限分离**
   - LSPosed Hook: 系统级权限
   - PhantomAPI: 应用级权限
   - 通过文件 IPC 解耦

3. **数据隔离**
   - 每次读取后删除文件
   - 防止数据泄露
